# Getting Started

Add beautiful notifications to your React app with [voby-toast](https://github.com/timolins/voby-toast).

### Install with Yarn

```sh
yarn add voby-toast
```

### Install with NPM

```sh
npm install voby-toast
```

## Basic usage

```jsx
import toast, { Toaster } from 'voby-toast';

const notify = () => toast('Here is your toast.');

const App = () => {
  return (
    <div>
      <button onClick={notify}>Make me a toast</button>
      <Toaster />
    </div>
  );
};
```
